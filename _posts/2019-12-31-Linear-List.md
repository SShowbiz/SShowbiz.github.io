## Linear List

Linear List는 선형구조를 가지는 가장 기본적인 자료 형태이다.

Linear List의 ADT는 다음과 같다.

-  type array[size]
-  int list_size
-  void insert(type element, int position)
-  void delete(type element, int position)
-  int list_size()
